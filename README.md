# SepsisGuard

## Overview
SepsisGuard is an AI-powered clinical decision support system that predicts sepsis risk in ICU patients and provides explainable, evidence-based treatment recommendations.

## Data
PhysioNet Sepsis Challenge 2019, MIMIC-III clinical notes, and public sepsis treatment guidelines.  
Raw and processed data are not included in this repository.

## Tech Stack
Python, PyTorch/TensorFlow, Weaviate, SentenceTransformers, Google Gemini, Streamlit.
